About

This tool was developed by Sathish kumar G under the supervision of Professor Arnab Dutta (in collaboration with Professor Debirupa Mitra) of BITS Pilani, Hyderabad Campus.

This webapp can be used to predict Wound Healing Peptide against the given peptide sequence.

This webapp is also capable of accepting input as CSV file containing the peptide sequences as one column. A new csv file conatining the predicted value of the properties as other columns will be automatically downloaded.



The format of output are as follows:

  • An error message will be displayed if an invalid sequence is entered.


The input requirements are as follows -

  • The entered peptide sequence should be in upper case only.

  • It should contain only the 20 naturally occuring amino acids

  • Example: GLFDVIKKVASVIGLASP

  • For uploading a CSV file, the file should only contain sequences as first column without any headers or index numbers.
